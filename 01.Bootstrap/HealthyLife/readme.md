# Oefening 02: HealthyLife 
## Bootstrap Layout & Content

Je krijgt een HTML-bestand zonder Bootstrap-classen. Gebruik Bootstrap om de pagina op te maken zoals in de voorbeeldscreenshots :

* screenshots/desktop.png
* screenshots/mobiel.png

Voeg Bootstrap toe aan het html bestand en probeer daarna de screenshot na te maken. 
Begin met de desktop versie.

Je moet o.a. gebruiken:

* Containers, rows en columns om de layout te structureren. Gebruik als breakpoint voor grotere schermen -md-. 
* Typografische klassen (display-, fw-, lead, text-muted, text-center, …)
* Kleur- en achtergrondklassen
* Afbeeldingsklassen (img-fluid, rounded, rounded-circle, shadow-sm, …)
* Marges en padding (mb-*, mt-*, py-*, px-*), gebruik eerder mb-* dan mt-* en me-* dan ms-*
